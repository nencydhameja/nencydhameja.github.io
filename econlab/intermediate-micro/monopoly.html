Monopoly

The setup

A single firm faces the entire market demand curve for its product. There are no competitors to undercut it. The firm picks one number — quantity \(q\) (or equivalently, price \(p\), since they pin each other down via the demand curve) — to maximize profit.

That’s the whole monopoly model. The geometry below shows everything: the demand curve, the marginal revenue curve, the marginal cost curve, the profit-maximizing quantity, and the deadweight loss relative to perfect competition.

The key insight: marginal revenue is below price

For a competitive firm, selling one more unit adds exactly \(p\) to revenue (you’re a price taker; selling more doesn’t move the price). So marginal revenue equals price.

For a monopolist, selling one more unit forces the price down on every unit. So marginal revenue is the price you get on the new unit minus the revenue you lose on all the previous units:

\[ \text{MR}(q) = p(q) + q \cdot p'(q). \]

Since \(p'(q) < 0\) (downward-sloping demand), \(\text{MR}(q) < p(q)\). The MR curve sits below the demand curve at every quantity.

With a linear demand \(p(q) = a - bq\):

\[ \text{MR}(q) = a - 2bq. \]

Same intercept as demand, twice the slope. That’s the rule worth memorizing.

Solving for \(q^*\)

The monopolist maximizes \(\pi(q) = p(q) \cdot q - C(q)\). The first-order condition is

\[ \text{MR}(q) = \text{MC}(q). \]

With linear demand \(p(q) = a - bq\) and constant marginal cost \(c\):

\[ \boxed{\; q^* = \frac{a - c}{2b}, \qquad p^* = \frac{a + c}{2}. \;} \]

Compare to perfect competition, where price equals marginal cost (\(p = c\)) and so \(q^{\text{comp}} = (a - c)/b\). The monopolist produces half the competitive quantity and charges a higher price. The gap between the two is the deadweight loss:

\[ \text{DWL} = \tfrac{1}{2} \cdot (q^{\text{comp}} - q^*) \cdot (p^* - c) = \frac{(a - c)^2}{8 b}. \]

That little triangle is the social cost of monopoly. The simulation below lets you watch it shrink and grow.

library(shiny)

ui <- fluidPage(
  titlePanel("Monopoly vs. Competition"),

  sidebarLayout(
    sidebarPanel(
      width = 4,
      sliderInput("a", "Demand intercept (a):",
                  min = 50, max = 200, value = 100, step = 10),
      sliderInput("b", "Demand slope (b):",
                  min = 0.5, max = 5,   value = 1.5, step = 0.1),
      sliderInput("c", "Marginal cost (c):",
                  min = 0,  max = 80,  value = 20,  step = 5),
      hr(),
      checkboxInput("show_comp", "Show competitive outcome for comparison",
                    value = TRUE),
      checkboxInput("show_dwl", "Shade the deadweight loss triangle",
                    value = TRUE),
      htmlOutput("readout")
    ),
    mainPanel(
      width = 8,
      plotOutput("monop_plot", height = "520px")
    )
  )
)

server <- function(input, output, session) {

  results <- reactive({
    a <- input$a; b <- input$b; c <- input$c
    q_mon  <- (a - c) / (2 * b)
    p_mon  <- a - b * q_mon
    q_comp <- (a - c) / b
    p_comp <- c
    profit <- (p_mon - c) * q_mon
    cs_mon  <- 0.5 * (a - p_mon) * q_mon
    cs_comp <- 0.5 * (a - p_comp) * q_comp
    dwl     <- 0.5 * (q_comp - q_mon) * (p_mon - c)
    list(a = a, b = b, c = c,
         q_mon = q_mon, p_mon = p_mon,
         q_comp = q_comp, p_comp = p_comp,
         profit = profit, cs_mon = cs_mon, cs_comp = cs_comp, dwl = dwl)
  })

  output$monop_plot <- renderPlot({
    r <- results()

    q_seq <- seq(0, r$a / r$b, length.out = 400)
    p_d  <- r$a - r$b * q_seq
    p_mr <- r$a - 2 * r$b * q_seq
    p_mr[p_mr < 0] <- NA

    xmax <- r$a / r$b
    ymax <- r$a * 1.05

    par(mar = c(4.2, 4.5, 1, 1))
    plot(NA, xlim = c(0, xmax), ylim = c(0, ymax),
         xlab = "Quantity (q)", ylab = "Price (p)", main = "")

    # MC curve (horizontal at c).
    abline(h = r$c, col = "#7f8c8d", lwd = 2, lty = 2)
    text(xmax * 0.95, r$c, "MC", pos = 3, col = "#7f8c8d", cex = 0.95)

    # Demand and MR.
    lines(q_seq, p_d,  col = "#185FA5", lwd = 2.5)
    lines(q_seq, p_mr, col = "#c0392b", lwd = 2, lty = 2)
    text(q_seq[10], p_d[10],  "Demand", pos = 4, col = "#185FA5", cex = 0.95)
    text(q_seq[10], p_mr[10], "MR",     pos = 4, col = "#c0392b", cex = 0.95)

    # Deadweight-loss triangle.
    if (isTRUE(input$show_dwl)) {
      polygon(c(r$q_mon, r$q_comp, r$q_mon),
              c(r$p_mon, r$c,      r$c),
              col = adjustcolor("#e67e22", 0.30), border = NA)
      text(mean(c(r$q_mon, r$q_comp)),
           (r$p_mon + r$c) / 2,
           "DWL", col = "#a04000", cex = 0.95, font = 2)
    }

    # Monopoly outcome.
    segments(r$q_mon, 0,      r$q_mon, r$p_mon, col = "#27ae60", lwd = 1.5, lty = 3)
    segments(0,       r$p_mon, r$q_mon, r$p_mon, col = "#27ae60", lwd = 1.5, lty = 3)
    points(r$q_mon, r$p_mon, pch = 19, col = "#27ae60", cex = 1.7)
    text(r$q_mon, r$p_mon,
         labels = sprintf(" (%.1f, %.1f)", r$q_mon, r$p_mon),
         pos = 4, col = "#27ae60", cex = 0.95)

    # Competitive outcome.
    if (isTRUE(input$show_comp)) {
      points(r$q_comp, r$p_comp, pch = 19, col = "#8e44ad", cex = 1.6)
      text(r$q_comp, r$p_comp,
           labels = sprintf(" (%.1f, %.1f)", r$q_comp, r$p_comp),
           pos = 4, col = "#8e44ad", cex = 0.95)
    }

    legend("topright",
           legend = c("Demand", "MR", "MC",
                      "Monopoly (q*, p*)",
                      if (isTRUE(input$show_comp)) "Competition (q^c, c)" else NULL,
                      if (isTRUE(input$show_dwl))  "Deadweight loss"      else NULL),
           col    = c("#185FA5", "#c0392b", "#7f8c8d", "#27ae60",
                      if (isTRUE(input$show_comp)) "#8e44ad" else NULL,
                      if (isTRUE(input$show_dwl))  adjustcolor("#e67e22", 0.5) else NULL),
           lwd    = c(2.5, 2, 2, NA,
                      if (isTRUE(input$show_comp)) NA else NULL,
                      if (isTRUE(input$show_dwl))  NA else NULL),
           lty    = c(1, 2, 2, NA,
                      if (isTRUE(input$show_comp)) NA else NULL,
                      if (isTRUE(input$show_dwl))  NA else NULL),
           pch    = c(NA, NA, NA, 19,
                      if (isTRUE(input$show_comp)) 19 else NULL,
                      if (isTRUE(input$show_dwl))  15 else NULL),
           bty = "n", cex = 0.85)
  })

  output$readout <- renderUI({
    r <- results()
    HTML(sprintf(paste(
      "<div style='margin-top:10px;font-size:13px;line-height:1.7;'>",
      "<b>Monopoly:</b> q* = %.1f, p* = %.1f, profit = %.1f<br>",
      "<b>Competition:</b> q = %.1f, p = %.1f<br>",
      "<b>Consumer surplus (monopoly):</b> %.1f<br>",
      "<b>Consumer surplus (competition):</b> %.1f<br>",
      "<b>Deadweight loss:</b> <span style='color:#a04000;font-weight:bold;'>%.2f</span>",
      "</div>"
    ),
    r$q_mon, r$p_mon, r$profit,
    r$q_comp, r$p_comp,
    r$cs_mon, r$cs_comp, r$dwl))
  })
}

shinyApp(ui, server)

Things to try.

  • Raise MC (\(c\)) to meet the demand curve at \(q = q^*\). Monopoly profit shrinks to zero and so does the DWL — the monopolist can’t price above MC anymore.
  • Lower \(b\) (flatter demand). The DWL grows fast. Flatter demand means consumers are more sensitive; the monopolist’s distortion costs more total surplus.
  • Compare profit and DWL across settings. The monopolist captures part of what would have been consumer surplus and destroys some — the DWL is the destroyed part. Both shrink as the demand curve gets steeper (more inelastic).
  • Set \(c = 0\). Monopoly produces half the competitive quantity (\(q^* = a/(2b)\), \(q^{\text{comp}} = a/b\)) and the DWL is \(a^2/(8b)\).

The Lerner index and elasticity

Re-arrange the monopolist’s first-order condition \(\text{MR}(q) = \text{MC}(q)\) using the elasticity of demand \(\varepsilon \equiv \frac{dq}{dp} \cdot \frac{p}{q}\):

\[ \frac{p^* - \text{MC}}{p^*} = -\frac{1}{\varepsilon}. \]

This is the Lerner index. It says the markup as a fraction of price equals the reciprocal of demand elasticity.

  • Very elastic demand (\(\varepsilon\) large in absolute value) → small markup.
  • Inelastic demand (\(\varepsilon\) near 1 in absolute value) → big markup.
  • A monopolist will never operate where \(|\varepsilon| < 1\) — they could raise price, sell less, and earn more revenue with no extra cost.

Did you know?

  • The “monopolist halves the quantity” result is a feature of linear demand. With constant-elasticity demand, the markup ratio \(p^*/\text{MC}\) is constant — but quantity is not exactly half. The general statement is the Lerner index.
  • First-degree price discrimination — the monopolist charges every consumer their willingness to pay — eliminates the deadweight loss. Quantity returns to the competitive level. The catch is that all the surplus goes to the firm; no consumer surplus left.
  • Modern antitrust analysis usually doesn’t try to estimate the DWL directly. Instead, it looks at whether a merger or business practice raises markups (via the Lerner index) or restricts entry. The geometry below is the conceptual scaffolding, not the empirical method.
